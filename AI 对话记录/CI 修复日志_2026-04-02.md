# GitHub Actions CI 修复日志

## 问题发现与修复

### 问题 1: Gameplay.tscn 解析错误
**错误信息**:
```
ERROR: Parse Error: Parse error. [Resource file res://scenes/Gameplay.tscn:79]
```

**原因**: Godot 4.x 中 `PackedVector2Array` 的语法变更
- Godot 3.x: `PackedVector2Array(Vector2(0, 0), Vector2(720, 0))`
- Godot 4.x: `PackedVector2Array([Vector2(0, 0), Vector2(720, 0)])`

**修复**: 更新 Gameplay.tscn 第 79 行
```diff
- points = PackedVector2Array(Vector2(0, 0), Vector2(720, 0))
+ points = PackedVector2Array([Vector2(0, 0), Vector2(720, 0)])
```

**提交**: `97ea1e8 fix: correct PackedVector2Array syntax in Gameplay.tscn for Godot 4.x`

## 常见问题排查

### Godot CI 容器问题
如果 `barichello/godot-ci:4.6.1` 容器有问题，可以尝试：
- `barichello/godot-ci:4.6` (最新 4.6.x)
- `barichello/godot-ci:4.6.2` (更新版本)

### Android 构建失败
Android 构建需要额外配置：
1. 确保容器包含 Android SDK
2. 设置 `ANDROID_HOME` 环境变量
3. 接受 SDK 许可证
4. 配置 Gradle

### 导出预设名称
确保工作流中的导出预设名称与 `export_presets.cfg` 中的一致：
- "Windows Desktop"
- "Linux/X11"
- "Web"
- "Android"
